# AI Advertisements

## 1. Инструкция по запуску проекта

### Требования
- Node.js 20+
- Docker и Docker Compose

### Запуск через Docker Compose

```bash
# Клонировать репозиторий
git clone https://github.com/3Lord3/AI-Advertisements
cd ai-advertisements

# Запустить все сервисы
docker compose up -d
```

После запуска будут доступны:
- Frontend: http://localhost:5180
- Backend: http://localhost:8081
- Ollama API: http://localhost:11434

### Первичная настройка Ollama

При первом запуске модель `llama3.2:1b` загрузится автоматически при первом запросе.

## 2. Обзор функциональности

### Основные возможности
- Реализованы страницы просмотра списка объявлений, просмотра объявления, редактирования объявления
- Реализованы сортировка, фильтрация и поиск объявлений. Поиск использует механизм debounce для оптимизации запросов
- Реализовано вычисление рыночной стоимости товара при помощи ИИ на основе схожих объявлений, с учётом всех характеристик, названия и описания
- Реализована генерация описания с помощью ИИ на основе данных объявления (Из-за очень слабой модели результаты не очень см. раздел "Компромиссы")
- Реализована поддержка переключения темы

### Стек технологий
- **Frontend**: React 19, TypeScript, Vite, TanStack Query, Zustand, shadcn/ui
- **AI**: Ollama (llama3.2:1b)
- **DevOps**: Docker, Docker Compose

## 3. Компромиссы
1. Так как моё устройство - ноутбук без дискретной видеокарты, вся работа выполнена со слабой моделью `llama3.2:1b` и ограниченными настройками Ollama, с соответствующим качеством генерации.
2. В бэкенде изменена конфигурация сервера для работы в Docker и в ответ API добавлены поля, ожидаемые от бекенда.
3. Для лучшего эффекта работы ИИ добавлено множество похожих объявлений с немного изменёнными характеристиками.
